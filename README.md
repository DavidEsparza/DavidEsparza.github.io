# Personal Portfolio Website

An interactive and modern personal portfolio website built with React, featuring animated components and a dynamic user interface.

## Coming Soon
### Videogame Data Analysis Showcase (Blue Prince)
A dedicated section featuring data analysis and visualizations from videogame datasets. This section will demonstrate data science skills through interactive charts, statistics, and insights derived from gaming data.

*This feature is currently under development and will be added in a future update.*

## Technologies Used
- React
- CSS3 with custom styling
- Particles.js for background effects
- i18n for internationalization

## Available Scripts

In the project directory, you can run:

### `yarn start`

Runs the app in the development mode.\
Open [http://localhost:3000](http://localhost:3000) to view it in your browser.

### `yarn build`

Builds the app for production to the `build` folder.\
The build is optimized for best performance.

### `yarn test`

Launches the test runner in the interactive watch mode.
